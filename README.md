# Biometric-Based Login System (HCI Group 3)

[cite_start]A secure authentication system designed using **Human-Computer Interaction (HCI)** principles[cite: 1, 5]. [cite_start]This project replaces traditional passwords with biometric data to improve security and user experience[cite: 3, 4].

## 🚀 Key Features
* [cite_start]**Biometric Authentication:** Fingerprint or facial recognition for secure access[cite: 51, 52].
* [cite_start]**Student Dashboard:** Organized layout for viewing profiles, schedules, and academic updates[cite: 25, 27, 33].
* [cite_start]**Admin Management:** Tools to monitor login attempts and manage student accounts[cite: 34, 35, 62].
* [cite_start]**Secure Storage:** Encrypted biometric data and user information.

## 🛠️ Tech Stack
* **Language:** PHP (OOP)
* **Frontend:** HTML5, Bootstrap 5
* **Backend:** Firebase (Connection pending)
* **Version Control:** GitHub

## 📂 OOP Structure
The system is built using an Object-Oriented approach for modularity:
- `User`: Base class for identity.
- [cite_start]`Student`: Handles dashboard and academic data[cite: 25, 32].
- [cite_start]`Admin`: Handles system logs and account control[cite: 36, 40].
- [cite_start]`DatabaseManager`: The bridge for secure data storage[cite: 48, 56].
